# Python File Organizer

A Python application that automatically sorts files into categorized folders based on file type.

## Features

- Organizes files automatically
- Creates folders if they do not exist
- Supports images, documents, PDFs, audio, and video files
- Simple and easy to use

## Technologies

- Python
- os
- shutil

## Future Improvements

- Graphical User Interface (GUI)
- Duplicate file detection
- Date-based sorting
